Ripple Notify

A simple and lightweight notification library

Documentation GitHub

Installation

npm i @codiify/ripple-notify

Usage

You need a ripple.js to your project root folder to customize the notification globally. File name must be ripple.js and the content should be like below:

export const RippleConfig = {
    timeout: 3000,
    animationSpeed: '0.5s',
    duration: 500,
    colorful: false,
    icon: true,
    position: 'top-right',
    colors: {
        success: {
            backgroundColor: '#ECFDF3',
            color: '#008A2E',
            borderColor: '#D3FDE5'
        },
        error: {
            backgroundColor: '#FFF0F0',
            color: '#E60000',
            borderColor: '#FFE0E1'
        },
        info: {
            backgroundColor: '#F0F8FF',
            color: '#0973DC',
            borderColor: '#D3E0FD'
        },
        warning: {
            backgroundColor: '#FFFCF0',
            color: '#DC7609',
            borderColor: '#FDF5D3'
        },
        default: {
            backgroundColor: '#FFFFFF',
            color: '#171717',
            borderColor: '#EDEDED'
        }
    }
};

Types

You can customize the type of toast you want to render, and pass an options object as the second argument.

ripple.default('Event has been created.')
ripple.success('You have achieved your goal üèÜ')
ripple.info('Please provide more information.')
ripple.warning('Warning! This action is irreversible.')
ripple.error('Oops! Something went wrong.')

Position

You can customize the position of the notification. Available options are

ripple.default('Event has been created.')
ripple.success('You have achieved your goal üèÜ')
ripple.info('Please provide more information.')
ripple.warning('Warning! This action is irreversible.')
ripple.error('Oops! Something went wrong.')
ripple.success('You have achieved your goal üèÜ')

Colored

If you want to show the colorful notification, you can set it to true. Default is false.

Icon

If you don't want to show the icon, you can set it to false. Default is true.

ripple.success('You have achieved your goal üèÜ')
